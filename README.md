# 🤖 Jarvis AI

A simple AI assistant web application with a chat interface, built using Node.js for the backend and a basic frontend (HTML, CSS, JS).

---

## 🚀 Features

- 💬 Chat interface
- ⚙️ Backend API using Node.js
- 🎨 Simple frontend UI
- 🔗 Easy to run locally

---

/project

└── README.md

## 🛠️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/0b1t07/ULTRON.git  
cd ULTRON

---

### 2. Install dependencies

npm install

---

### 3. Run the backend server

node server.js

Or (if you add a start script in package.json):

npm start

---

### 4. Run the frontend

Open index.html in your browser.
